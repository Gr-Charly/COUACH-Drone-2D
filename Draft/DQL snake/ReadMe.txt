Avant de lancer le projet, assurez-vous d’avoir installé les bibliothèques nécessaires :

pip install pygame torch matplotlib numpy ipython



Pour lancer la simulation :

Lancer le fichier agent.py 




Explication :

agent.py : logique de l’IA (états, actions, mémoire, apprentissage, lancement du jeu)

game.py : L'environnement via Pygame (affichage, collisions, score)

helper.py : affichage du graphique des scores

model.py : modèle de réseau de neurones et entraîneur

arial.ttf : police utilisée pour le score
